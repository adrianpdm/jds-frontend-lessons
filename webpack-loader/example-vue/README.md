# Webpack Loader within Vue Project

Note: Project is scaffolded using VueCLI.

This lessons meant to demonstrate the use of webpack loader in transform *.svg into Vue component using [github.com/egoist/svg-to-vue-component](github.com/egoist/svg-to-vue-component).

You can inspect recent Webpack config by running these command
```bash
vue inspect > webpackconfig

# config can then be checked on ./webpackconfig
```
